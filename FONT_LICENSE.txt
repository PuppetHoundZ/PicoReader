Scope: LICENSE.md (pure MIT text) covers this project's own source code --
main.py, epub_engine.py, mini_jpeg.py, mux_launch.sh, gutenberg_fetch.py,
native_image.py, jw_fetch.py, and PLUGIN_TEMPLATE.py. It does NOT cover the
bundled fonts below, which ship under the separate, permissive license
described in this file.

Bundled fonts: DejaVu Sans Condensed -- assets/font.ttf (Regular),
assets/font-bold.ttf (Bold), assets/font-italic.ttf (Oblique),
assets/font-bolditalic.ttf (Bold Oblique).

License summary follows below (bitstream-vera license, a permissive font
license predating and distinct from OFL 1.1). Full canonical text:
https://dejavu-fonts.github.io/License.html

Source: the fonts-dejavu-core Debian/Ubuntu package (version 2.37-8),
upstream https://dejavu-fonts.github.io/. Embedded name-table version
field confirms "Version 2.37" on the regular weight. Not modified in
any way from the original release.

License summary: DejaVu's own changes are public domain; the
underlying Bitstream Vera base is permissively licensed (free to use,
copy, merge, publish, distribute -- the one real restriction is that
the font file itself can't be sold standalone, separate from a larger
software package, which doesn't apply here since PicoReader is free).
Renaming is required if the font is modified and redistributed under
the Bitstream/Vera name specifically -- not a concern here since these
files are unmodified.

Inter (OFL 1.1), Liberation Sans (OFL 1.1), and DejaVu Sans (regular
width, same license family as this font) were all bundled at earlier
points in this project's history (see main.py's FONT_PATHS comment for
the full switch history) but are no longer shipped as of v0.1.152.
Liberation Sans is still referenced by filename as an on-device
system-path fallback if the bundled DejaVu Sans Condensed files are
ever missing -- that path loads the OS's own installed copy under
muOS, not a file this project ships, so no separate license obligation
applies to that fallback reference.
